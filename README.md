CSE425 Neural Network Project

***Unsupervised Learning Project: VAE for Hybrid Language Music Clustering**

This repository contains the implementation of the CSE425 course project
on music data analysis using neural networks and clustering techniques.
## Tasks Implemented

Easy Task
- Variational Autoencoder (VAE) for feature extraction using audio features (MFCCs)
- K-Means clustering on latent representations
- Cluster visualization using t-SNE and UMAP
- Baseline comparison using PCA + K-Means
- Evaluation using Silhouette Score and Calinski-Harabasz Index

Medium Task
- Multimodal learning using audio and lyrics features
- Feature fusion and clustering analysis

Hard Task
- Advanced VAE variants (e.g., Beta-VAE / Conditional VAE)
- Improved latent representation learning and clustering

## Tools and Libraries
- Python
- TensorFlow / Keras
- Scikit-learn
- Google Colab

## How to Run

1. Open any `.ipynb` file from the `SRC/` folder in Google Colab.
2. Upload the dataset when prompted.
3. Run all cells sequentially.
